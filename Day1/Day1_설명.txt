1. Anaconda 설치

2. Anaconda Prompt 실행
	- 아나콘다가 관리하는 root 경로 : C:\Users\Administrator
	- 우리가 사용 할 작업 폴더로 이동
	>

3. (base)
	: Anaconda가 관리하는 대표 가상환경
	: 이곳에는 라이브러리 설치를 하지 않습니다.

	- (base) 가상공간에 설치된 라이브러리 목록 확인
		>conda list

	- Python Version 확인
		>python --version

	- Anaconda 명령어 : conda 라는 이름으로 명령을 시작합니다.
		-- conda 버전 확인
			>conda --version

	- Anaconda에 생성된 가상환경(가상공간) 목록 확인하기
		>conda env list

	- Jupyter Notebook 프로그램 에디터 열기
		--> 내가 작업 할 작업 폴더 경로 확인 후 아래 실행
			>jupyter notebook

	- (base) 가상공간과 웹브라우저 jupyter notebook editor와 연결하기 위해서는 커널(kernel) 연결을 통해서 파일을 생성해야 가상공간의 라이브러리를 사용할 수 있습니다.
		(base -> Python3 커널을 통해서 연결되어 있음)

	- Jupyter Notebook에서 사용하는 Python 파일 확장자 : *.ipynb
		--> 순수한 Python 파일 확장자 *.py로 변환하려면 별도로 변환하여 저장

4. 새로운 가상환경 생성하여 진행합니다.
	- 우리가 별도로 생성 할 가상환경(가상공간)은 (base) 가상공간에서 수행
