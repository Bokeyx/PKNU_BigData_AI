1. 아나콘다 설치
2. Anaconda Prompt 실행
 - 아나콘다가 관리하는 root 경로 : C:\Users\Administrator
 - 우리가 사용할 작업폴더로 이동 
    >cd C:\Users\Administrator\pk_202503

3. (base) : 아나콘다가 관리하는 대표 가상환경(가상공간)
            : 이곳에는 라이브러리 설치를 하지 않습니다.

   - (base) 가상공간에 설치된 라이브러리 목록 확인
     >conda list
 
   - 파이썬 버전 확인하기
     > python --version
 
   - 아나콘다 명령어 : conda 라는 이름으로 명령을 시작 합니다.
     -- conda 버전 확인
        >conda --version

   - 아나콘다에 생성된 가상환경(가상공간) 목록 확인하기
     >conda env list

   - 주피터 노트북(jupyter notebook) 프로그램 에디터 열기
     --> 내가 작업할 작업폴더 경로 확인 후 아래 실행
     >jupyter notebook

   - (base) 가상공간과 웹브라우저 주피터 노트북 에디터와 연결하기 위해서는
     커널(kernel) 연결을 통해서 파일을 생성해야 가상공간의 라이브러리를
     사용할 수 있습니다. 
     (base 공간은 -> Python3 커널을 통해서 연결되어 있음)

  - 주피터 노트북에서 사용하는 파이썬 파일 확장자 : *.ipynb
    --> 순수한 파이썬 파일 확장자 *.py로 변환하려면 별도로 변환하여 저장

4. 새로운 가상환경(가상공간) 생성하여 진행합니다.
   - 우리가 별도고 생성할 가상환경(가상공간)은 (base) 가상공간에서 수행
